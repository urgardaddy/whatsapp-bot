# Clone this project

```bash
> git clone https://github.com/Restaa/bot
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd bot
> bash install.sh
```

### Usage
```bash
> npm start
```

### Settings
```json
{
	"prefix": ".",
	"ownerNumber": "62858xxxxx",
	"apiKey": "Your ApiKey here"
}
```
